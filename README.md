# Maquette MUI Experience

Vitrine 100% front-end bâtie avec React 18, TypeScript et MUI v5. Toutes les données sont mockées en mémoire afin de tester rapidement les parcours et les thèmes.

## Démarrage
```bash
npm install
npm start
```

## Pages incluses
- Dashboard avec KPI, graphique Recharts et flux d'activité
- DataGrid complet (tri, filtres, export CSV)
- Formulaire avancé (React Hook Form + Zod + DatePicker)
- Kanban drag & drop (persisté en localStorage)
- Galerie de composants MUI

## Thèmes
Palette switcher accessible via l'icône Palette (Modern Dark, Light Corporate, Glassmorphism, Retro 90s).
